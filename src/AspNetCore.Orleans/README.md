# Mississippi.AspNetCore.Orleans

This project provides ASP.NET Core integration adapters backed by Orleans, including:

- `IDistributedCache` - Orleans-backed distributed cache implementation
- `ITicketStore` - Authentication ticket storage using Orleans
- `IOutputCacheStore` - Output cache storage using Orleans
- `HubLifetimeManager` - SignalR hub lifetime management with Orleans
- Additional adapters as needed (e.g., `ISessionStore`, `IXmlRepository`)

## Status

This project is under development and ready for incremental implementation.
